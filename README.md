* Typescript:

`npm add -D typescript`

* Converter TS em JS:

`npm add -D sucrase`

* Automatic restart:

`npm add -D nodemon`

* Executar um script:

`npm run-script dev`

* análise de código estática para identificar padrões problemáticos:

`npm add -D eslint @typescript-eslint/parser @typescript-eslint/eslint-plugin`

`npx eslint --init`
